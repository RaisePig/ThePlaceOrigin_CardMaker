# 桌游卡牌制作系统

一个精美的响应式桌游卡牌制作系统，支持手机和电脑屏幕自适应。

## 功能特性

- 🎨 **人物立绘上传**：支持图片上传，可自由拖动调整立绘位置
- 🏷️ **编号管理**：自定义卡牌编号
- 👤 **人物信息**：设置人物姓名和卡牌等级（SR/SSR/UR）
- ⚔️ **技能系统**：最多添加2个技能，支持主动/被动选择
- 💥 **绝技系统**：最多添加1个绝技，支持主动/被动选择
- 📊 **属性设置**：力量、敏捷、生命（0-9整数）和种类（文字）

## 技术栈

- React 18
- TypeScript
- Tailwind CSS
- Vite

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 使用说明

1. **上传立绘**：点击"点击上传图片"按钮选择图片文件
2. **调整位置**：上传后，在预览区域拖动图片可以调整立绘位置
3. **填写信息**：在右侧表单中填写编号、姓名等信息
4. **设置等级**：点击SR/SSR/UR按钮选择卡牌等级
5. **添加技能**：点击"添加技能"按钮，最多可添加2个技能
6. **添加绝技**：点击"添加绝技"按钮，最多可添加1个绝技
7. **设置属性**：填写力量、敏捷、生命（0-9）和种类

## 响应式设计

- 手机端：单列布局，表单在上方，预览在下方
- 电脑端：双列布局，预览在左侧，表单在右侧
